Settings Menu

  • Pressing ⋮ or the Options button of your device will bring up the Options Menu.
  • Haptic Feedback: toggle vibration upon pressing a button.
    • When activated, the device will vibrate when a valid button is pressed.
    • If a button is pressed and no vibration occurs, it means that the action corresponding to that button is invalid in the current state. The display will also briefly flash to signal an invalid input.
  • Radians/Degrees:select unit to be used for trigonometric calculations.
  • User Manual:view this manual.

Display

  • The display zone is divided into two lines:
    • The top line shows the expression as the user composes it. It also displays past expressions when using the Previous (▲) and Next (▼) buttons.
    • The bottom line displays the evaluation of the current expression when the = button is pressed.
  • The display will flash briefly when you try to enter an invalid input (such as two decimal points in a single number).

Resetting the Display

  • Press C to clear all the inputs/results and reset the calculator to the initial state. This operation is allowed any time during the calculation.
  • The C button affects neither History nor Memory.

Composing and Evaluating Expressions

  • As you compose an expression, it gets displayed on the top line of the display.
    • At any time, you can modify the expression. Refer to the Making Corrections During Input section.
  • When you are ready to evaluate the expression, press the = button.
    • If the expression is valid, the result appears on the bottom line of the display
    • If the expression is invalid, an error message is displayed on the bottom line.
  • After being evaluated, the expression is still displayed on the top line, but is no longer editable.
    • To modify it and re-evaluate it, press the ▲ button to activate the expression from History. Refer to the History section for more information.

Arithmetic Calculations

  • An arithmetic operator (+, ÷, ×, -) must be preceded and followed by an operand.

Scientific Functions

  • When calling a scientific function (log10(x), xy, sin(x), √x) ,the function is displayed and an opening parenthesis is automatically added. After entering the desired number, you must manually add closing parenthesis using the ) button.

      • Sine Function

    • The calculation of the trigonometric function (sin(x)) evaluates radians by default but can be configured to evaluate degrees via the Settings Menu.
      • Example: sin(x), π, ), = ⇒ sin(π) = 0

      Logarithm on the Base of 10

      Example: log10(x), 1, 0, ), = ⇒ log10(10) = 1

      Square Root

      Example: √x, 4, ), = ⇒ √4 = 2

      Power Function

      Example: 2, ×, 3, = ⇒ 23 = 8

Parentheses

  • An opening parenthesis is added automatically in scientific functions that require it.
  • Otherwise, parentheses can be entered manually using the ( and ) .
  • A numerical operand followed or preceded by another numerical operand between parentheses implies multiplication.
    • Example:
    2(2+4) ⇒ 2×(2+4) = 12
    (2+4)2 ⇒ (2+4)×2 = 12
  • An operand will multiply a constant (π or e) or a scientific function even without parentheses.
    • Example:
    2π ⇒ 2×π = 6,283185307179631
    2Log10(10) = 2×Log1010 = 2
  • A discrepancy between the number of opening and closing parentheses will result in an error when the = button is pressed.

Making Corrections During Input

  • An expression can be modified at any time by using the cursor buttons ◀ ▶ .
    • Position the cursor where a correction is needed and use the ⌫ button to remove the needed characters.
    • You can then input a new operand/operator as needed.
  • The resulting expression must be valid in order to be evaluated.
    • Eternity enforces valid input to the left of the cursor. Therefore, it is possible to accidentally enter invalid characters.
      • Example:
      1..|2 is rejected.
      1.|.2 is accepted but is invalid.

Memory

  • The result of the latest evaluation of an expression is always stored automatically. To recall it, press the Answer button Ans.
  • To store a value for later reuse, press the Memory Store button MS.
    • The MS function will store the value that is currently displayed in the bottom line of the screen. In order to save a number, you must first evaluate it.
      • Example:
      After pressing C , the bottom line of the screen displays ‘0.0’
       
      1, 2, 3, = MS ⇒ stores 123 in memory
       
      1 2 3 MS ⇒ stores 0.0 in memory
       
  • To recall the stored value, press the Memory button M
  • The content of a memory slot (Ans or M) can be used while editing an expression.
    • The screen will display either [M] or [ANS] as an operand in the expression.
    • For syntactic purposes, [M] and [ANS] are treated like literal operands (i.e. like π or e), meaning they will be multiplied by a preceding operand even if not enclosed in parentheses.

History

  • Eternity keeps a history of all expressions that have previously been evaluated. These expressions can be browsed using the ▲ ▼ buttons.
  • The expression currently being viewed can be modified and re-evaluated. Refer to section Making Corrections During Input for more information.
  • When an expression is re-evaluated, it becomes the most recent expression in the history.

Exporting a Result or an Expression

  • A result or an expression can be copied to the clipboard of your Android device by long-pressing the display window.
